# Feature Spec: UI 전면 재편 — 보라/파랑 테마 + 로그인 우선 대시보드

**Feature ID**: 002  
**Branch**: `002-ui-redesign`  
**Date**: 2026-04-19  
**Author**: jeongho

---

## Overview

현재 프론트엔드는 디자인 요소가 거의 없는 기본 회색 레이아웃이다.
보라색과 파란색을 주 테마로 하는 세련되고 미래지향적인 UI로 전면 재편하며,
사이트 접속 시 로그인 화면이 먼저 표시되고 로그인 후에는 관심 종목 카드 대시보드로 진입하는 UX 흐름으로 변경한다.

---

## User Stories

### US-001: 미인증 방문자 → 로그인 화면 진입
**As a** 사이트 방문자,  
**I want** 사이트에 접속했을 때 세련된 로그인 화면이 표시되기를,  
**So that** 서비스의 첫인상이 전문적이고 미래지향적으로 느껴진다.

**Acceptance Criteria**:
- 비로그인 상태로 `/` 접속 시 `/login` 페이지로 리다이렉트된다
- `/login` 페이지는 보라→파랑 그라디언트 배경을 가진다
- 서비스 로고, 소개 문구, Google 로그인 버튼, Kakao 로그인 버튼이 표시된다
- 이미 로그인된 경우 `/dashboard`로 바로 리다이렉트된다

### US-002: 로그인 후 대시보드 진입
**As a** 인증된 사용자,  
**I want** 로그인 후 내 관심 종목들을 카드 형태로 볼 수 있는 대시보드로 이동하기를,  
**So that** 중요 정보에 빠르게 접근할 수 있다.

**Acceptance Criteria**:
- 로그인 성공 시 `/dashboard`로 리다이렉트된다
- 대시보드는 관심 종목 카드를 반응형 그리드(2~3열)로 표시한다
- 각 카드는 종목명, 현재가, 등락률, AI 투자 권유 총점을 포함한다
- AI 분석 결과가 있는 경우 단기/중기/장기 점수를 추가로 표시한다
- 카드 클릭 시 해당 종목 상세 페이지(`/stock/[ticker]?market=[market]`)로 이동한다
- 관심 종목이 없으면 안내 문구와 종목 검색 링크를 표시한다
- 로딩 중에는 스켈레톤 카드를 표시한다

### US-003: 보라/파랑 테마 디자인 시스템 적용
**As a** 사용자,  
**I want** 전체 UI가 보라색과 파란색을 기반으로 한 세련되고 미래지향적인 디자인을 가지기를,  
**So that** 앱 사용 경험이 현대적이고 전문적으로 느껴진다.

**Acceptance Criteria**:
- 페이지 배경: 다크 슬레이트(`slate-950`) 계열
- 주요 강조: 보라→파랑 그라디언트 (`purple-600` → `blue-500`)
- 카드: 반투명 글래스모피즘 (`bg-white/5 backdrop-blur-md border-white/10`)
- 상승/하락: 에메랄드/로즈 포인트 컬러
- 기존 모든 페이지(watchlist, scores, stock, chat)에 새 테마 적용

### US-004: 네비게이션 재디자인
**As a** 사용자,  
**I want** 네비게이션 바가 새 테마에 맞게 재디자인되기를,  
**So that** 전체적인 시각적 일관성이 유지된다.

**Acceptance Criteria**:
- Nav bar: 반투명 다크 배경 + `backdrop-blur`
- 로고: 그라디언트 텍스트 (보라→파랑)
- 현재 페이지 활성 링크 표시 (언더라인 + 밝은 텍스트)
- 로그인된 경우 사용자 아바타/이름 + 로그아웃 버튼 표시
- 모바일 반응형 지원

---

## Success Criteria

1. 비로그인 방문자가 `/`에 접속하면 즉시 `/login` 화면이 표시된다
2. OAuth 로그인 성공 후 `/dashboard`에서 관심 종목 카드를 확인할 수 있다
3. 모든 페이지가 보라/파랑 다크 테마로 일관성 있게 표시된다
4. 기존 기능(종목 검색, AI 분석, 채팅, 점수 비교)이 새 디자인 아래 정상 동작한다

---

## Out of Scope

- 백엔드 API 변경 없음 (순수 프론트엔드 작업)
- 새로운 기능 추가 없음 (디자인 재편 및 UX 흐름 변경만)
- 화이트/라이트 테마 옵션 없음 (다크 테마 단일 적용)
- Framer Motion 등 추가 애니메이션 라이브러리 도입 없음
